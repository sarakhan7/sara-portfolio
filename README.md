# Sara Khan — Personal Portfolio

This is my personal portfolio website, I am a computer science major currently seeking a software engineering internship. The site showcases selected projects, technical skills, and includes a working contact form. Designed to be responsive, accessible, and visually distinct.

## Features

- Custom-designed light and dark themes
- Responsive layout across all devices
- Project cards with interactive hover effects
- Functional contact form powered by Formspree
- Smooth scrolling and animated section transitions
- Deployed with Vercel

## Built With

- Next.js (App Router)
- Tailwind CSS
- Lucide React (icon library)
- Formspree (contact form handling)
- ShadCN UI + Radix UI (component styling and accessibility)
